# Roadmap del Stack de Reconocimiento

##  Etapa actual
- Integración de módulos existentes en un stack coherente.
- Documentación de arquitectura y metodología.
- Cloudflare real origin detection (completado).

## Próximas funcionalidades
- Detección avanzada de WAF con fingerprint JA3/JA4.
- Screenshot automation (Playwright / Selenium).
- Reportes en JSON y HTML personalizables.
- DNS brute recursive enumeration.
- Wordlist inteligente generada dinámicamente.
- Censys/Shodan enrichment automático.

##  Visión a futuro
Desarrollar un framework de Recon modular, expandible y capaz de integrarse con
pipelines modernos de ataque/defensa, pensado tanto para aprendizaje como para
entornos reales.
